# Телеграм-бот для библиотек Петроградской стороны

**Семестр: 3**
**Тип:** командный проект

**Описание:**
Бот для Telegram, который предоставляет информацию о графике работы, адресах и доступных книгах в библиотеках Петроградского района. Реализован парсинг данных с сайта библиотек и интеграция с Telegram API. Работа выполнялась в команде.
